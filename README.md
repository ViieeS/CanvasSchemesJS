# Canvas Schemes JS

## Example
```HTML
<ul id="parts">
<li><span class="part_number" data-part-number="1">1</span>Name</li>
...
</ul>

<canvas id="scheme"></canvas>

<script>
new Scheme(
'scheme',
'parts',
'sheme.png',
'[{"sLabel":"1","label_x1":158,"label_y1":165},{"sLabel":"1","label_x1":49,"label_y1":540}]'
);
</script>
```
